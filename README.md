# AWS-S3-Static-Site
This project demonstrates how to host a static website using Amazon S3 (Simple Storage Service). The goal of this project is to deploy a simple HTML-CSS based website on AWS Cloud and make it publicly accessible using S3 static website hosting features. The repository contains necessary website file of HTML and CSS. It demonstrates cloud deployment skills using Amazon Web Services (AWS).

Key Highlights & AWS Skills:

1. AWS S3 Static Website Hosting:

The website is hosted on AWS S3, showcasing the ability to deploy a fully static website in the cloud.

Configured public access and bucket policies for secure, accessible hosting.



2. Cloud Deployment Workflow:

Created an S3 bucket, enabled static website hosting, and uploaded project files (index.html and assets).

Used AWS Management Console for deployment and testing.



3. Scalability & Reliability:

Leveraged AWS’s highly available and scalable storage infrastructure for hosting.

Demonstrates understanding of cloud-native hosting solutions for small projects.



4. Cost-Effective Hosting:

Utilized AWS Free Tier to deploy the project with zero cost.

Demonstrates knowledge of optimizing cloud resources for small-scale deployments.



5. Cloud Security Basics:

Configured bucket policies to allow public read access only for static website files.

Understood basic AWS security principles for S3 hosting.



6. Frontend Skills:

Built a responsive, mobile-friendly website with clean UI/UX design.

Interactive features like “Learn More” alerts demonstrate frontend interactivity.



7. Project Showcase for Portfolio:

Demonstrates end-to-end cloud-based website deployment for recruiters or clients.

Emphasizes ability to combine web development and cloud deployment skills.


Purpose:
This project is a portfolio-ready AWS project, showcasing the ability to design, deploy, and manage static websites on the AWS cloud.
